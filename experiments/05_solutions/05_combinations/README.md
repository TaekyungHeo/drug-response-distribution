# Combinations

Do solutions combine additively, or do they exploit the same signal?

| # | Experiment | Question | Key result |
|---|-----------|---------|------------|
| 01 | `01_moa_x_kshot/` | Within-MoA + K-shot factorial | EGFR: within-MoA alone ≈ combined; ERK: K-shot dominates (K=50 r=0.813) |
| 02 | `02_lincs_x_moa/` | LINCS + within-MoA 2×2 factorial | LINCS does not improve per-drug r; within-MoA improves per-drug r only; effects are on different axes |
